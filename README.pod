NAME
    FOEGCL - utilities for the Friends of the East Greenbush Community
    Library.

VERSION
    Version 0.01

SYNOPSIS
    This distribution provides command line tools and libraries to assist
    the Friends of the East Greenbush Community Library simplify manual
    tasks.

  COMMAND LINE TOOLS
    compare-registered-voters
        compare-registered-voters - Compare Friends' voter registration
        statuses against the voter registration roll

  LIBRARIES
    FOEGCL::CSVProvider
        FOEGCL::CSVProvider - an extendable object that iterates over the
        rows of a CSV file.

    FOEGCL::ItemStore
        FOEGCL::ItemStore - an extendable object that stores items with a
        set of similar characteristics.

    FOEGCL::Logger
        FOEGCL::Logger - an extendable object that provides output logging
        services.

    FOEGCL::GOTV::____
        FOEGCL::GOTV::_____ - a set of modules that assist the
        compare-registered-voters program.
